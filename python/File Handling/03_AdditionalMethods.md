# Additional File Handling Methods

## Read:
1) `readlines()` : returns all lines in the file as a list of lines.
2) `read()` : returns the content of the entire file as a string.
3) `readline()` : returns one line from the file.

Write:
1) `write()` : writes specified text to a file.
2) `writelines()`: writes the elements of a list as seperate lines to a file.

Cursor Methods:
1) `tell()` : returns the current position of the file handle cursor.
2) `seek()` : used to change position of file handle cursor.
