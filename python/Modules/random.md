# Module: Random

The random module allows us to attain randomized values out of the blue.
> import random

## Methods:
1) `randint(a,b)`: returns a random integer 'n' that belongs to the range a <= n <= b.
2) `random()`: returns a random floating decimal point that belongs to the range 0 <= n < 1.
3) `choice(iterable)`: returns a random element from the iterable.
